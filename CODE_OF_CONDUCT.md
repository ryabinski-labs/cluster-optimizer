# Code of Conduct

This project expects respectful, constructive participation.

## Expected Behavior

- Be direct, factual, and respectful.
- Keep discussion focused on Cluster Optimizer, Kubernetes cost analysis, documentation, and user impact.
- Share enough evidence for maintainers to reproduce issues without exposing private cluster or cloud data.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing another person's private information without permission.
- Posting secrets, kubeconfigs, exploit details, or sensitive logs in public issues or pull requests.
- Repeatedly derailing technical discussion after maintainers ask to refocus.

## Enforcement

Maintainers may edit or remove comments, close issues, block users, or take other moderation actions needed to keep the project usable. To report a conduct issue, contact `cigan1@gmail.com`.
