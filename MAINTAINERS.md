# Maintainers

This file is the human-readable source of truth for project maintainers. Repository permissions should mirror these entries through explicit collaborator access or a dedicated maintainer team.

Maintainers can merge pull requests to the default branch after required CI/status checks pass. A blanket approving review is not required by default, and non-maintainers must not be able to merge even if checks pass.
